# <center>nonebot-plugin-getbapics</center>
## 使用方法
* 输入```/setu+[tag]+[x|X|*][num]+量词```使用
* tag自选， ```x``` ```X``` ```*```可选，num可选，量词可选```张``` ```个``` ```份```
* ~已实现空格自由~
### 例子
* ```/setu```
* ```/setu 4```
* ```/setu 美游 x4张```
* ```/setu x4张```
* ```/setu 美游 x4个```
* ```/setu 美游 4张```
* ```/setu 美游 4```
